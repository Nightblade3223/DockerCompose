# DockerCompose

## Included services

- **NPMPlus** (`zoeyvid/npmplus`) has been added as a reverse proxy manager service.
- NPMPlus upstream feature list: https://github.com/ZoeyVid/NPMplus?tab=readme-ov-file#list-of-new-features
- **Plex** (`linuxserver/plex`) has been added as an additional media server using the same media library mounts as Jellyfin.
